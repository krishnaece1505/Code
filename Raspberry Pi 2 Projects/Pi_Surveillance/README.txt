//---------------------------------------------------------------------------------
   A home surveillance project using Raspberry Pi 2 and Pi-NoIR camera. The Pi
captures pics from the camera every minute and uploads them to my Dropbox account. 
Dropbox-Uploader script by Andrea Fabrizi has been used for uploading the pics
to cloud. Crontab command is used to schedule the picam_surveillance python 
script to be executed every minute.
